# Hybrid_anomaly_detection_digital_twins

A Python-based anomaly detection system that combines **DBSCAN** and **Isolation Forest** to identify abnormal patterns in digital twin sensor data. The application provides analysis, anomaly visualization.

## Features
- data preprocessing
- DBSCAN-based anomaly detection
- Isolation Forest outlier detection
- Hybrid anomaly prediction
- data visualization
- Feature contribution analysis

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Tkinter

## Run the Project
```bash
pip install -r requirements.txt
python app.py
